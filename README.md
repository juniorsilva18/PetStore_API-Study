# PetStore_API-Study
Pt- Portugues. 
Projeto de estudo voltado para testes de API utilizando o Swagger Petstore.

Neste repositório, desenvolvi cenários automatizados cobrindo operações básicas de CRUD, utilizando Java, RestAssured e Cucumber (BDD). A estrutura foi organizada separando camadas de steps e lógica, buscando uma abordagem mais próxima de projetos reais de automação.

O objetivo é evoluir na área de QA, consolidar boas práticas de testes e ganhar mais experiência prática com automação de APIs.

EN - English. 
Study project focused on API testing using Swagger Petstore.

In this repository, I developed automated test scenarios covering basic CRUD operations using Java, RestAssured and Cucumber (BDD). The structure was organized by separating steps and logic layers, aiming to follow a more realistic automation approach.

The goal is to grow as a QA professional, strengthen testing best practices and gain more hands-on experience with API automation.
